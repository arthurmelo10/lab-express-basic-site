This is an example of a web application using the framework Express

Express is a framework, in other words, it is a group of pre set-structures that allows the developers to build web pages, without all the logic involved.

The objective here is to create a web application, based on Express, in which the students from Ironhack's WebDev BootCamp must choose some character or some person and write about some aspects oh their lives, and work
